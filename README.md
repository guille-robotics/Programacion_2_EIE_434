# 💻 EIE 434 — Programación 2

Repositorio oficial del curso **EIE 434 - Programación 2**.

En este repositorio se publicará el material utilizado durante el semestre, incluyendo **guías de clases prácticas**, **guías de laboratorio** y **códigos de referencia** utilizados durante el curso.

---

## 📚 Contenido del repositorio

Aquí encontrarás material utilizado durante el desarrollo del curso:

### 📄 Guías en PDF
- Guías de ejercicios utilizadas en clases prácticas
- Guías de laboratorio
- Actividades y desafíos del curso

### 💻 Códigos de ejemplo y soluciones
- Implementaciones en **Python**
- Implementaciones en **C**
- Soluciones de referencia para los ejercicios propuestos

---

## 🧠 Objetivo del repositorio

El objetivo de este repositorio es que los estudiantes puedan:

- Revisar los **ejercicios trabajados en clases**
- Consultar **soluciones de referencia**
- Practicar programación de manera autónoma
- Tener acceso organizado a todo el **material práctico del curso**

---

## 🛠 Lenguajes utilizados en el curso

Durante el curso se trabajará principalmente con:

🐍 **Python**  
⚙️ **C**

Además se utilizarán herramientas comunes en desarrollo de software:

- Git
- GitHub
- Visual Studio Code
- Terminal / línea de comandos

---

## 🚀 Recomendaciones para los estudiantes

Para aprovechar mejor el material del repositorio se recomienda:

✔ Intentar resolver los ejercicios **antes de revisar las soluciones**  
✔ Ejecutar los códigos y **modificarlos para experimentar**  
✔ Utilizar los ejemplos como base para **crear sus propios programas**
